# 🔎 ElasticSearch Mapping Prompt

Design optimized search indexes and complex queries for ElasticSearch or OpenSearch.

## Prompt

```text
Act as a Search Engineer. Design an ElasticSearch index mapping for [ENTITY, e.g., Products, Log events].

Fields: [e.g., title, description, category, price, created_at]

Include:
1. Analyzer configuration (Tokenizers, Filters) for search relevance.
2. Field types (Keyword vs Text vs Dense Vector).
3. Mapping for N-grams or Autocomplete.
4. A sample "Search Query" that includes filtering, aggregation, and scoring.
```
