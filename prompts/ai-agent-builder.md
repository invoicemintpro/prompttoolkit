# 🤖 AI Agent Builder Prompt

Design complex, multi-step AI agents with tools, memory, and clear objectives.

## Prompt

```text
Act as an AI Agent Architect. I want to build an autonomous agent for [OBJECTIVE, e.g., researching competitors, automating customer support].

Toolbox: [e.g., Google Search, Database Read/Write, API Executor]
Environment: [e.g., Python, LangChain, AutoGPT]

Define the Agent's:
1. Role & Identity.
2. Step-by-step reasoning process (Chain of Thought).
3. Tool Usage logic (When and how to use tools).
4. Error handling and loop prevention.
5. Success criteria.
```

## How to use
1. Clearly define what "done" looks like for the agent.
2. List all external APIs or tools the agent has access to.
