# 🚩 Feature Flag Strategy Prompt

Design robust feature flag implementations for staged rollouts and A/B testing.

## Prompt

```text
Act as a Senior Product Engineer. Design a feature flag system for [FEATURE NAME].

Requirements:
1. Target Audience segments (e.g., Internal users, 10% of production).
2. Fallback logic if the flag service is unreachable.
3. Clean-up strategy to avoid technical debt.
4. Tracking/Analytics integration for the rollout.
```
