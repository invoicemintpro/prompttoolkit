# 🏗️ System Design Prompt

Design scalable, robust, and efficient system architectures for modern applications.

## Prompt

```text
Act as a Principal Solutions Architect. I need a high-level system design for [APP NAME/DESCRIPTION].

Requirements:
1. Expected Traffic: [e.g., 100k DAU, high read/write ratio]
2. Core Features: [FEATURE 1, FEATURE 2, FEATURE 3]
3. Budget/Timeline Constraints: [Optional]

Deliverables:
- Recommended Tech Stack (Frontend, Backend, Database, Caching).
- Architecture Diagram Description (Components and their interactions).
- Data Flow Analysis.
- Scalability Strategy (Horizontal vs Vertical, Load Balancing).
- Potential Bottlenecks and how to mitigate them.
```

## How to use
1. Be specific about your app's core value proposition.
2. Define your expected load to get tailored infrastructure advice.
