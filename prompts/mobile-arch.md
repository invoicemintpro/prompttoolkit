# 📱 Mobile App Architecture Prompt

Design scalable and maintainable architectures for iOS (Swift) or Android (Kotlin) apps.

## Prompt

```text
Act as a Mobile Lead Engineer. Propose an architecture (MVVM, VIPER, or Clean Architecture) for a [iOS/Android] app that handles [CORE FEATURE, e.g., real-time chat, offline data sync].

Include:
1. Layer separation (Data, Domain, Presentation).
2. Dependency Injection strategy.
3. Reactive programming vs Structured Concurrency recommendation.
4. Local storage strategy (CoreData, Room, Realm).
5. Networking layer (URLSession, Retrofit) with retry logic.
```
