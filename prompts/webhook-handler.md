# 🪝 Webhook Handler Prompt

Securely implement and test incoming webhooks from services like Stripe, GitHub, or Shopify.

## Prompt

```text
Act as a Backend Lead. Design a secure webhook handler for [SERVICE NAME].

Requirements:
1. Signature verification logic to prevent spoofing.
2. Idempotency handling (duplicate events).
3. Asynchronous processing (immediate 200 OK response).
4. Error logging and retry handling.
5. Provide a sample implementation in [LANGUAGE].
```
