# 🔄 Data Migration Strategy Prompt

Plan and execute complex data migrations between different databases or versions.

## Prompt

```text
Act as a Data Engineer. I need a migration plan from [SOURCE DB] to [TARGET DB].

Context: [e.g., Moving 10M user records, zero-downtime required]

Deliverables:
1. Migration Stages (Extract, Transform, Load).
2. Schema Mapping details.
3. Validation and Reconciliation strategy.
4. Rollback plan in case of failure.
5. Performance optimization for the migration script.
```
