# 🐚 Shell Script Expert Prompt

Generate powerful Bash or PowerShell scripts for automation and system administration.

## Prompt

```text
Act as a Senior DevOps Engineer. Write a [Bash/PowerShell] script to automate [TASK].

Task: [e.g., Backup all databases, compress them, and upload to S3]

Requirements:
1. Include error handling and logging.
2. Ensure the script is idempotent.
3. Use variables for configuration.
4. Add comments for each logical block.
```
