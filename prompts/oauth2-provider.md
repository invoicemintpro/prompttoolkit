# 🔑 OAuth2 Provider Design Prompt

Design and implement custom OAuth2/OpenID Connect providers for your platform.

## Prompt

```text
Act as an Identity Architect. Design an OAuth2 provider flow for [APP NAME].

Requirements:
1. Supported Grant types (Authorization Code, Client Credentials).
2. Token generation logic (JWT vs Opaque).
3. Scopes and Permissions definition.
4. Security: PKCE (Proof Key for Code Exchange) implementation.
5. Provide a sample /authorize and /token endpoint structure.
```
