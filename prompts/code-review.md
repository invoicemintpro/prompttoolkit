# 🧑‍💻 Code Review Prompt

Use this prompt to get a comprehensive, professional code review focusing on performance, security, and maintainability.

## Prompt

```text
Act as a Senior Software Engineer with expertise in [LANGUAGE/FRAMEWORK].
Review the following code for:
1. Potential bugs or edge cases.
2. Performance bottlenecks.
3. Security vulnerabilities (OWASP Top 10).
4. Code readability and adherence to [STYLE GUIDE, e.g., PEP 8, Airbnb].
5. Suggestions for better architectural patterns.

Provide your feedback in a structured format with "Issue", "Explanation", and "Suggested Fix".

Code to Review:
---
[INSERT CODE HERE]
---
```

## How to use
1. Replace `[LANGUAGE/FRAMEWORK]` with your stack (e.g., React, Go, Python).
2. Replace `[STYLE GUIDE]` if applicable.
3. Paste your code snippet in the designated section.
