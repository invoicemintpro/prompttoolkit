# ⚖️ Load Testing Plan Prompt

Design and execute performance load tests to ensure your application can handle scale.

## Prompt

```text
Act as a Performance QA Engineer. Design a load testing plan for [APP/ENDPOINT].

Target Load: [e.g., 5,000 concurrent users]
Tool: [e.g., k6, JMeter, Locust]

Deliverables:
1. Test Scenarios (Happy path, Stress test, Soak test).
2. Key Metrics to track (Response time, Error rate, CPU/Memory).
3. Test Script logic (e.g., k6 script).
4. Analysis guide for identifying bottlenecks from results.
```
