# 🌿 Git Branching Strategy Prompt

Design effective branching and merging workflows (GitFlow, Trunk-Based, etc.) for your team.

## Prompt

```text
Act as a DevOps Lead. Propose a Git branching strategy for a team of [NUMBER] developers working on [APP TYPE].

Requirements:
1. Support for rapid feature development and hotfixes.
2. Integration with CI/CD.
3. Code review and approval process.
4. Recommendation: Trunk-based vs GitFlow.
```
