# 📖 API Documentation (Swagger) Prompt

Generate complete OpenAPI/Swagger specifications for your API endpoints.

## Prompt

```text
Act as a Technical Writer. Generate a Swagger (OpenAPI 3.0) specification for the following endpoint.

Endpoint: [e.g., POST /v1/payments]
Parameters: [e.g., amount, currency, source_id]
Success Response: [e.g., 201 Created with payment_id]
Error Responses: [e.g., 400 Bad Request, 401 Unauthorized]

Output:
- Full YAML or JSON specification.
- Examples for request and response bodies.
```
