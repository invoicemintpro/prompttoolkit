# 📧 Email Writer Prompt

Write professional, clear, and effective emails for common developer scenarios like feature requests, bug reports, or client updates.

## Prompt

```text
Act as a Professional Communications Specialist. Write a [TONE, e.g., formal, friendly, urgent] email regarding:

Subject: [CONTEXT, e.g., Delayed feature deployment, Request for API access]
Key Points to Include:
- [POINT 1]
- [POINT 2]
- [POINT 3]

Goal: [e.g., Get approval, provide an update, schedule a meeting]

The email should be concise, professional, and have a clear Call to Action (CTA).
```

## How to use
1. Define the tone and subject.
2. List the bullet points you want the AI to expand upon.
3. State your ultimate goal for the communication.
