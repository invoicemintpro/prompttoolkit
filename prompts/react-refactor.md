# ⚛️ React Component Refactor Prompt

Refactor messy React components into clean, performant, and reusable functional components.

## Prompt

```text
Act as a Senior Frontend Developer. Refactor the following React component.

Code:
---
[INSERT REACT COMPONENT]
---

Task:
1. Extract logic into custom Hooks where appropriate.
2. Improve performance using useMemo/useCallback/React.memo.
3. Add Prop-Types or TypeScript interfaces.
4. Ensure the component follows the Single Responsibility Principle.
5. Provide the refactored version and a summary of changes.
```
