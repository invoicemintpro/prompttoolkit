# 🎨 UI/UX Improvement Prompt

Get actionable feedback on your UI designs and UX flows to improve user engagement and accessibility.

## Prompt

```text
Act as a Senior UI/UX Designer. Analyze the following [WEBSITE/APP] screen description or image.

Context: [e.g., E-commerce Checkout Page, SaaS Dashboard]
Goal: [e.g., Increase conversion, reduce churn]

Review for:
1. Visual Hierarchy and Layout.
2. Accessibility (WCAG 2.1 compliance).
3. Call to Action (CTA) effectiveness.
4. Micro-copy and Clarity.

Provide 5 specific, high-impact changes to improve the user experience.
```

## How to use
1. Describe your layout or paste a link/screenshot if using a vision-capable model.
2. State the primary action you want users to take on this screen.
