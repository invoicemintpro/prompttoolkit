# 🛠️ Refactoring Legacy Code Prompt

Safely refactor legacy codebases to improve maintainability without breaking functionality.

## Prompt

```text
Act as a Senior Software Engineer. I need to refactor this legacy code snippet.

Code:
---
[INSERT LEGACY CODE]
---

Goal: [e.g., Improve readability, add type safety, remove technical debt]

Deliverables:
1. Analysis of current "smells" or issues.
2. Step-by-step refactoring plan.
3. Refactored version of the code.
4. Suggested unit tests to ensure no regression.
```
