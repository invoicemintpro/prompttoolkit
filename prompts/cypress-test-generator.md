# 🧪 Cypress Test Generator Prompt

Generate end-to-end (E2E) tests for your web application using Cypress.

## Prompt

```text
Act as a QA Automation Engineer. Generate Cypress E2E tests for the following user flow.

User Flow: [DESCRIBE FLOW, e.g., Login -> Add to Cart -> Checkout]
Target URL/Elements: [DESCRIBE SELECTORS OR PAGE STRUCTURE]

Requirements:
1. Ensure the tests are robust (use data-cy if possible).
2. Include assertions for each critical step.
3. Add comments explaining the logic.
4. Output the full Cypress spec file.
```
